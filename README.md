# Titanic
# Model Created for Predicting Titanic Survival. 

The predictive model uses both Logistic Regression and Random Forest to determine the survival rate of the test data. The data was obtained from Kaggle https://www.kaggle.com/competitions/titanic/data. Below are some of the graphs created during the data evaluation process. The goal was to train the models with the training data and use the test data to predict the outcome of the new data loaded into the model.
